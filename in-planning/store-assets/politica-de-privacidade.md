## 2. Sobre este documento

Esta Política de Privacidade descreve como o **TL em Par** coleta, usa, armazena e protege os dados pessoais dos usuários do aplicativo e do site. Ao utilizar nossos serviços, você concorda com as práticas descritas aqui, em conformidade com a **Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018)**.

---

## 3. Dados que coletamos

### 3.1 Dados fornecidos por você
- **Nome completo** — para identificação na plataforma
- **Endereço de e-mail** — para login e comunicações
- **CPF** — para identificação fiscal na assinatura
- **Telefone** — opcional, para contato
- **Foto de perfil** — opcional, enviada por você

### 3.2 Dados de uso
- Histórico de benefícios resgatados (restaurante, data, horário)
- Avaliações de restaurantes

### 3.3 Dados de pagamento
Os pagamentos são processados exclusivamente pela plataforma **Stripe** (stripe.com). O TL em Par **não armazena** dados de cartão de crédito. Consultamos apenas o status da assinatura (ativa, inativa, cancelada) junto à Stripe.

### 3.4 Dados do dispositivo (somente no aplicativo)
- Token de notificação push (FCM/APNs) — para envio de alertas sobre benefícios e novos restaurantes
- Câmera — usada exclusivamente para leitura de QR Code nos restaurantes parceiros; nenhuma imagem é armazenada

---

## 4. Como usamos os dados

| Finalidade | Base legal (LGPD) |
|-----------|-------------------|
| Autenticação e acesso à conta | Execução de contrato (art. 7º, V) |
| Validação de benefícios via QR Code | Execução de contrato (art. 7º, V) |
| Envio de notificações push sobre benefícios | Legítimo interesse / consentimento (art. 7º, II e IX) |
| Comunicações por e-mail (confirmações, alertas) | Execução de contrato (art. 7º, V) |
| Melhoria da plataforma (dados de uso agregados) | Legítimo interesse (art. 7º, IX) |

---

## 5. Compartilhamento de dados

Não vendemos nem cedemos seus dados a terceiros para fins comerciais. Compartilhamos apenas com:

- **Stripe Inc.** — processamento de pagamentos e gestão de assinatura
- **Resend** — envio de e-mails transacionais
- **Cloudinary** — armazenamento de fotos de perfil
- **Google Firebase** — envio de notificações push (Android)
- **Apple APNs** — envio de notificações push (iOS)

Todos os fornecedores operam com políticas de privacidade compatíveis com a LGPD e/ou GDPR.

---

## 6. Retenção dos dados

Seus dados são mantidos enquanto sua conta estiver ativa. Após o cancelamento da conta, os dados pessoais são excluídos em até **90 dias**, exceto quando houver obrigação legal de retenção (ex: dados fiscais por 5 anos conforme legislação tributária brasileira).

---

## 7. Seus direitos (LGPD)

Você pode, a qualquer momento, solicitar:

- **Confirmação** da existência de tratamento dos seus dados
- **Acesso** aos dados que temos sobre você
- **Correção** de dados incompletos ou desatualizados
- **Eliminação** dos seus dados
- **Portabilidade** dos seus dados
- **Revogação do consentimento**

Para exercer qualquer desses direitos, envie um e-mail para **contato@tlempar.com.br**.

---

## 8. Segurança

Adotamos medidas técnicas e organizacionais para proteger seus dados, incluindo:

- Comunicações criptografadas via HTTPS/TLS
- Senhas armazenadas com hash bcrypt
- Tokens de acesso JWT com expiração curta (15 minutos)
- Acesso ao banco de dados restrito à infraestrutura interna

---

## 9. Menores de idade

O TL em Par é destinado a maiores de 18 anos. Não coletamos intencionalmente dados de menores. Se você acredita que um menor forneceu dados à plataforma, entre em contato pelo e-mail acima para exclusão imediata.

---

## 10. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Em caso de mudanças relevantes, você será notificado por e-mail ou por aviso dentro do aplicativo. O uso continuado após a notificação implica aceitação das alterações.
